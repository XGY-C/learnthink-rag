# 人工智能导论 — 第3章 贝叶斯分类器

## 3.1 贝叶斯理论基础

### 3.1.1 贝叶斯定理

**贝叶斯定理**（Bayes' Theorem）是概率论中的基本定理，描述了在已知某些条件下某事件发生的概率。

公式：**P(A|B) = [P(B|A) × P(A)] / P(B)**

其中：
- **P(A|B)**：后验概率（Posterior），给定 B 发生时 A 的概率
- **P(B|A)**：似然度（Likelihood），给定 A 时 B 的概率
- **P(A)**：先验概率（Prior），A 在没有任何条件下的概率
- **P(B)**：证据因子（Evidence），B 的边缘概率

### 3.1.2 先验概率与后验概率

**先验概率**（Prior Probability）是在观测到任何数据之前，根据历史经验对事件发生概率的估计。

**后验概率**（Posterior Probability）是在观测到新数据之后，利用贝叶斯定理对先验概率进行修正后的结果。

举例：已知某种疾病在人群中的患病率为 1%（先验），检测准确率为 99%。如果某人检测为阳性，他实际患病的概率是多少？这就是贝叶斯定理的典型应用。

## 3.2 朴素贝叶斯分类器

### 3.2.1 基本思想

**朴素贝叶斯分类器**（Naive Bayes Classifier）基于贝叶斯定理，假设各特征之间**条件独立**。

对于给定样本 x = (x₁, x₂, ..., xₙ)，属于类别 Ck 的概率为：

**P(Ck | x₁,...,xₙ) ∝ P(Ck) × ∏ᵢ P(xᵢ | Ck)**

由于分母 P(x) 对所有类别相同，只需比较分子部分，选择后验概率最大的类别。

### 3.2.2 "朴素"假设的含义

"**朴素**"（Naive）指的是：假设所有特征在给定类别的条件下互相独立。

在实际中这个假设通常不成立，但朴素贝叶斯在实践中效果仍然很好的原因是：
1. 分类决策只依赖于后验概率的排序，而非精确值
2. 即使概率估计有偏差，只要排序正确，分类结果就是正确的
3. 对小样本数据有较好的泛化能力

### 3.2.3 三种常见变体

**高斯朴素贝叶斯**（GaussianNB）：
- 假设特征服从**正态分布**（高斯分布）
- 适用场景：特征为连续值，如身高、温度
- 参数估计：计算每个类别下各特征的均值 μ 和方差 σ²

**多项式朴素贝叶斯**（MultinomialNB）：
- 特征表示计数（如词频）
- 适用场景：**文本分类**、文档主题识别
- 典型应用：垃圾邮件过滤

**伯努利朴素贝叶斯**（BernoulliNB）：
- 特征为二元值（0/1，出现/不出现）
- 适用场景：短文本分类、情感分析

### 3.2.4 拉普拉斯平滑

**拉普拉斯平滑**（Laplace Smoothing）解决零概率问题——如果某个特征值在训练集中未出现，其概率会被估计为 0，导致整个后验概率为 0。

公式：**P(xᵢ | Ck) = (count(xᵢ, Ck) + α) / (count(Ck) + α × V)**

- α：平滑参数（α=1 为拉普拉斯平滑）
- V：特征可能的取值数量

## 3.3 半朴素贝叶斯分类器

**半朴素贝叶斯分类器**（Semi-Naive Bayes）放松了完全条件独立的假设，允许部分特征之间存在依赖关系。

常见策略：
- **SPODE**（Super-Parent ODE）：所有特征依赖同一个"超父"特征
- **TAN**（Tree-Augmented Naive Bayes）：特征间形成树形依赖结构
- **AODE**（Averaged ODE）：集成多个 SPODE 的结果

## 3.4 贝叶斯网络

**贝叶斯网络**（Bayesian Network）是一种概率图模型，用**有向无环图**（DAG）表示变量之间的条件依赖关系。

### 3.4.1 结构组成
- **节点**：表示随机变量
- **有向边**：表示变量之间的依赖关系（因果或影响）
- **条件概率表**（CPT）：每个节点在给定父节点取值条件下的概率分布

### 3.4.2 推理类型
- **因果推理**（Causal）：从原因推断结果
- **诊断推理**（Diagnostic）：从结果推断原因
- **解释推理**（Intercausal）：多个原因之间的交互

## 3.5 实战：朴素贝叶斯文本分类

### 3.5.1 文本分类流程

1. **文本预处理**：分词、去停用词
2. **特征构建**：词袋模型（Bag of Words）或 TF-IDF
3. **训练**：计算每个类别的先验概率 P(Ck) 和条件概率 P(w|Ck)
4. **预测**：对每条新文本，计算所有类别的后验概率，选择最大值

### 3.5.2 代码示例（Python）

```python
from sklearn.naive_bayes import MultinomialNB
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split

# 准备数据
texts = ["人工智能改变世界", "机器学习是人工智能的核心",
         "今天天气很好", "周末去哪里玩"]
labels = ["科技", "科技", "生活", "生活"]

# 向量化
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# 训练
clf = MultinomialNB(alpha=1.0)  # 拉普拉斯平滑
clf.fit(X, labels)

# 预测
new_text = ["深度学习的发展趋势"]
X_new = vectorizer.transform(new_text)
print(clf.predict(X_new))  # ['科技']
```

## 3.6 优缺点分析

**优点**：
- 理论基础坚实，分类效率高
- 对小规模数据表现好，不容易过拟合
- 天然支持增量学习，可逐条更新模型

**缺点**：
- 条件独立假设在实践中常被违反
- 对输入数据的表达形式敏感
- 需要先验概率，先验设定可能影响结果

## 3.7 本章小结

贝叶斯分类器是最经典的分类方法之一。朴素贝叶斯以"条件独立"假设换取计算效率，在实际文本分类任务中表现优异。贝叶斯网络则提供了更灵活的概率建模框架。
