## 二、计算题

### 2.1 朴素贝叶斯手算

已知训练集：

| 天气 | 温度 | 湿度 | 是否打球 |
|------|------|------|---------|
| 晴   | 高   | 高   | 否      |
| 晴   | 高   | 正常 | 否      |
| 阴   | 高   | 高   | 是      |
| 雨   | 低   | 高   | 是      |
| 雨   | 低   | 正常 | 是      |
| 阴   | 低   | 正常 | 否      |

**问题**：天气=阴、温度=低、湿度=正常时，是否打球？

解题步骤：
1. 计算先验概率 P(是)=3/6=0.5，P(否)=3/6=0.5
2. 计算条件概率：
   - P(阴|是) × P(低|是) × P(正常|是) × P(是) = (1/3) × (2/3) × (1/3) × 0.5
   - P(阴|否) × P(低|否) × P(正常|否) × P(否) = (1/3) × (0/3) × (1/3) × 0.5
3. 第二个式子出现 0，需用拉普拉斯平滑（α=1）
4. 比较后验概率，取最大类别

### 2.2 医疗诊断贝叶斯推理

某种疾病患病率为 0.1%，检测准确率为 99%（即患者检测阳性概率为 99%，健康人检测阴性概率为 99%）。某人检测为阳性，他真正患病的概率是多少？

**解题**：
- P(患病)=0.001, P(阳性|患病)=0.99
- P(健康)=0.999, P(阳性|健康)=0.01
- P(阳性)=0.99×0.001 + 0.01×0.999 = 0.00099 + 0.00999 = 0.01098
- **P(患病|阳性)=0.99×0.001/0.01098≈0.09=9%**

关键洞察：即使检测准确率 99%，阳性结果也有 91% 的可能来自健康人的假阳性！

## 三、编程实战

### 3.1 垃圾邮件分类

使用 sklearn 的 MultinomialNB 实现垃圾邮件分类：

```python
from sklearn.naive_bayes import MultinomialNB
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.pipeline import Pipeline
from sklearn.metrics import classification_report

# 示例数据
emails = [
    "免费领取优惠券，点击链接了解详情",       # spam
    "明天会议在3楼会议室，请准时参加",         # ham
    "恭喜您获得大奖，请回复手机号领取",         # spam
    "项目进度报告已发到邮箱，请查收",           # ham
    "限时特惠，错过今天再等一年",              # spam
    "本周五下午团建，请大家投票选地点",          # ham
]

labels = ["spam", "ham", "spam", "ham", "spam", "ham"]

# 构建 Pipeline
pipeline = Pipeline([
    ('tfidf', TfidfVectorizer(token_pattern=r'(?u)\b\w+\b')),
    ('clf', MultinomialNB(alpha=1.0)),
])

pipeline.fit(emails, labels)

# 测试
test = ["请您查收上季度项目报告"]
prob = pipeline.predict_proba(test)
print(f"垃圾邮件概率: {prob[0][1]:.2%}")
```
